# TestCode

A collaborative coding interview platform with real-time video, code execution, and session management.

## Features
- Real-time collaborative coding
- Video & voice calls
- Live chat
- Code execution engine
- LeetCode-style problems
- Session management
- Contest system

## Tech Stack
- Frontend: React + Vite
- Database: Supabase
- Styling: TailwindCSS + DaisyUI
- Real-time: Supabase Realtime

## Setup
1. Clone the repository
2. Install dependencies: `npm install --prefix frontend`
3. Configure environment variables (see `.env.example`)
4. Run the app: `npm run dev --prefix frontend`

## Deployment
Ready for deployment on Netlify. See `netlify.toml` for configuration.
